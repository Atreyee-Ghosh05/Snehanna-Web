# 🍔 Home Delivery App

## Description
This is a simple **Home Delivery App** built with HTML, CSS, and JavaScript.  
Users can browse menu items, place orders, and track delivery status in real-time.

## Features
- Browse menu with images and prices
- Add items to cart
- Place orders online
- Track delivery status
- Customize spice level 🌶️
